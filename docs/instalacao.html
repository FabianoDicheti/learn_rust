Instalação e Setup

A instalação do Rust é surpreendentemente simples graças ao rustup, o gerenciador oficial de versões do Rust. O rustup não instala apenas o compilador, mas todo um ecossistema de ferramentas poderosas.

Instalação com Rustup

Para instalar o Rust no Linux, macOS ou Windows com WSL, execute o seguinte comando no terminal:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Instalação no Windows

Para Windows nativo, baixe o instalador em rustup.rs e execute o arquivo .exe.

Durante a instalação:

  1. Escolha a instalação padrão (opção 1)
  2. Aguarde o download e instalação das ferramentas
  3. Feche e abra o terminal novamente

Verificando a Instalação

Após a instalação, verifique se tudo foi instalado corretamente:

$ rustc --version rustc 1.70.0 (90c541806 2023-05-31) $ cargo --version cargo 1.70.0 (ec8a8a0ca 2023-04-25)

Ferramentas Instaladas

O rustup instala automaticamente um conjunto completo de ferramentas:

cargo

Gerenciador de pacotes e sistema de build

O Cargo é seu melhor amigo no desenvolvimento Rust. Ele gerencia dependências, compila projetos, executa testes e muito mais.

rustc

Compilador do Rust

O compilador oficial que transforma seu código Rust em executáveis otimizados. Raramente usado diretamente - o Cargo cuida disso.

rustfmt

Formatador de código

Formata automaticamente seu código seguindo as convenções oficiais do Rust. Integra-se perfeitamente com editores.

clippy

Linter e analisador

Um "assistente" inteligente que aponta erros comuns, sugere melhorias e ensina boas práticas de Rust.

Configuração do Editor

Recomendação: VS Code + rust-analyzer

Para a melhor experiência de desenvolvimento, recomendamos:

  1. Instalar o Visual Studio Code
  2. Instalar a extensão rust-analyzer da Microsoft Store de extensões
  3. A extensão oferece: auto-complete, verificação de erros em tempo real, refatoração e muito mais

Mantendo o Rust Atualizado

O Rust tem um ciclo de lançamento de 6 semanas. Para manter suas ferramentas atualizadas:

# Atualizar todas as ferramentas do Rust rustup update # Verificar versões instaladas rustup show # Instalar uma versão específica (opcional) rustup install stable rustup install beta rustup install nightly

Comandos Úteis do Rustup

rustup update - Atualiza todas as ferramentas
rustup show - Mostra versões instaladas
rustup default stable - Define versão padrão
rustup --help - Ajuda completa
