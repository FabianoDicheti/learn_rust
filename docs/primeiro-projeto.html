Primeiros Passos

Agora que o Rust está instalado, vamos criar nosso Iniciando Um Projeto! O Cargo torna isso incrivelmente simples e já configura tudo que você precisa para começar a programar.

Criando um Novo Projeto

Use o comando cargo new para criar um novo projeto:

$ cargo new ola_rust Created binary (application) `ola_rust` package $ cd ola_rust

Dica

O Cargo cria automaticamente um projeto completo com estrutura de diretórios, arquivos de configuração e até um repositório Git!

Estrutura do Projeto

Vamos explorar o que o Cargo criou para nós:

Árvore de Diretórios

ola_rust/ ├── Cargo.toml # Manifesto do projeto ├── src/ │ └── main.rs # Código-fonte principal ├── target/ # Gerado ao compilar └── .git/ # Repositório Git

Arquivos Importantes

Cargo.toml - Configuração e dependências
src/main.rs - Ponto de entrada do programa
target/ - Arquivos compilados (gerado automaticamente)

Explorando o Cargo.toml

O arquivo Cargo.toml é o coração do seu projeto. Vamos ver o que ele contém:

[package] name = "ola_rust" version = "0.1.0" edition = "2021" # Dependências do projeto (inicialmente vazio) [dependencies]

Sobre o Cargo.toml

  • name: Nome do seu projeto
  • version: Versão atual (segue versionamento semântico)
  • edition: Edição do Rust (2021 é a mais recente)
  • dependencies: Bibliotecas externas que seu projeto usa

Seu Primeiro "Olá, Mundo!"

O Cargo já criou um programa "Olá, Mundo!" para nós. Vamos ver o conteúdo de src/main.rs:

fn main() { println!("Hello, world!"); }

Vamos personalizar nossa mensagem:

fn main() { println!("Olá, Rust!"); println!("Bem-vindo ao mundo da programação de sistemas!"); }

Compilando e Executando

O Cargo oferece várias maneiras de trabalhar com seu projeto:

cargo run

Compila e executa o projeto em um único comando. Ideal para desenvolvimento.

$ cargo run Compiling ola_rust v0.1.0 (/path/to/ola_rust) Finished dev [unoptimized + debuginfo] target(s) in 0.75s Running `target/debug/ola_rust` Olá, Rust! Bem-vindo ao mundo da programação de sistemas!
cargo check

Verifica se o código compila sem gerar executável. Muito mais rápido que cargo run.

$ cargo check Checking ola_rust v0.1.0 (/path/to/ola_rust) Finished dev [unoptimized + debuginfo] target(s) in 0.32s
cargo build

Apenas compila o projeto, gerando o executável em target/debug/.

$ cargo build Compiling ola_rust v0.1.0 (/path/to/ola_rust) Finished dev [unoptimized + debuginfo] target(s) in 0.75s
cargo build --release

Compilação otimizada para produção. Mais lenta, mas gera executáveis muito mais rápidos.

$ cargo build --release Compiling ola_rust v0.1.0 (/path/to/ola_rust) Finished release [optimized] target(s) in 1.23s

Entendendo a Função main

Vamos analisar nosso código linha por linha:

// Função principal - ponto de entrada do programa fn main() { // println! é uma macro (note o !) // Imprime texto na tela com quebra de linha println!("Olá, Rust!"); }
fn main() - Toda aplicação Rust começa aqui
println! - Macro para imprimir texto (note o !)
Chaves {} - Delimitam blocos de código
Ponto-e-vírgula ; - Termina declarações

Comandos Úteis do Cargo

# Criar novo projeto cargo new meu_projeto # Criar biblioteca em vez de aplicação cargo new minha_lib --lib # Limpar arquivos compilados cargo clean # Mostrar informações do projeto cargo metadata # Formatar código automaticamente cargo fmt # Executar linter (clippy) cargo clippy
